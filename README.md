​

파일경로에 생성된 log_xxxxx_xxxxx.csv파일을 선택하고

​

저장경로에  저장할 위치와 파일명 지정이 가능합니다. 

만약 저장위치와 파일명을 직접 지정하기 귀찮이면 `현재경로`로 지정하는 버튼이 있습니다. 

*파일경로를 입력하지 않거나 `현재 경로`를 클릭하지 않으면 이미 존재하던 그림에 덮어 씌워집니다. 

​
